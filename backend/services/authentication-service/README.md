# User service
Simple auth service for the user to log in an recieve a token

# Routes

## POST:/api/v1/users/
```
Create a new user
```
## POST:/api/v1/users/login/
```
Authorize the user and returns a token
```
## PUT:/api/va/users/:id
```
Updates the user's password
```
## DELETE:/api/va/users/:id
```
Deletes the user
```
